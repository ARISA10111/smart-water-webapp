# Smart Water Web Application

## Project Overview

Smart Water is a responsive web application designed to connect customers with trusted water vendors across Kenya. The platform provides information about water delivery services, company background, vendor details, pricing, and customer contact options.

The project aims to address challenges related to water accessibility by making it easier for households and businesses to locate reliable water suppliers and request water delivery services.

## Features

* Responsive multi-page website
* User-friendly navigation bar
* Home page with company introduction
* About page with mission, vision, and objectives
* Services page showcasing available water services
* Vendors page displaying registered water suppliers
* Pricing page with service costs
* Gallery page featuring water delivery operations
* Contact page with customer inquiry form
* JavaScript form validation
* Bootstrap-powered responsive design

## Technologies Used
