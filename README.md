# smart-water-webapp
Smart water for smart people

Smart Water is a responsive web application designed to connect customers with trusted water vendors across Kenya. The platform provides information about water delivery services and helps users learn more about the company and contact service providers.

This project is being developed as part of a Web Application Development assignment using HTML5, CSS3, Bootstrap 5, JavaScript, Git, and GitHub.
